# __ConvertX__


__This bot can turn your text into a voice :).It uses technologies such as:__ \
· Silero(neural network model)\
· gTTS\
· Microsoft voices

## System requirements
__OC__: Linux/Windows 7/8/10/11\
__Python Version__: 3.9\
__Hard Disk Memory__: 100MB+\
__RAM__: 1GB+\
__CPU__: ALL+\
__GPU__: If possible
